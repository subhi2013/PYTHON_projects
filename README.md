Tic Tac Toe

A SIMPLE TIC TAC TOE GAME BUILT USING PYTHON

Features:
  (1)Two-player gameplay
  (2)Input validation
  (3)Win detection
  (4)Row, column and diagonal checks
  (5)Draw detection
  (6)Replay option
Technologies Used:
  Python
How to Run:
step 1: Download the project.
step 2:Open a terminal.
step 3: Run:
          python tic_tac_toe.py
