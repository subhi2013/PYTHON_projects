📦 Inventory Management System

A Python-based Inventory Management System built using Python, Pandas, and CSV for efficient product management.

✨ Features:

➕ Add Product
📋 View Products
🔍 Search Product
✏️ Update Product
❌ Delete Product
⚠️ Low Stock Alert
💰 Sort Products by Price
📦 Sort Products by Stock
🏆 Highest Price Product
💵 Lowest Price Product
📊 Total Products
💲 Total Inventory Value

🛠️ Technologies Used
Python
Pandas
CSV File Handling
