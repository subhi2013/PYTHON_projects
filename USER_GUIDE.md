📊 Student Performance Analyzer

📖 About

This is a Python Data Analysis project that analyzes student marks from a CSV file using Pandas, NumPy, and Matplotlib.

🚀 Features

(1) Read student data from a CSV file
(2) Calculate Total Marks
(3) Calculate Average Marks
(4) Find the Topper
(5) Find the Lowest Scorer
(6) Calculate Class Average
(7) Assign Ranks
(8) Assign Grades
(9) Display Pass/Fail Status
(10) Generate Bar Chart
(11) Generate Pie Chart
(12) Export the final report to a new CSV file

🛠 Technologies Used
Python
Pandas
NumPy
Matplotlib

📂 Project Files
student_performance_analyzer.py
students_performance.csv
student_report.csv

▶️ How to Run

Install the required libraries:

pip install pandas numpy matplotlib

Run the program:

python student_performance_analyzer.py

📈 Output
Student Performance Report
Bar Graph of Average Marks
Pie Chart of Grade Distribution
Exported CSV Report
