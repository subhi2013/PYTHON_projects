# HOTEL MANAGEMENT SYSTEM

## DESCRIPTION

       This is a simple hotel management system developed using python. it stores guest details in a text file using file handling.

## FEAUTURES

      - Add guest
      - View guest
      - Search guest
      - Update guest
      - Check out guest
      - View available rooms
      - Generate bill

## TECHNOLOGIES USED
    - Python
    - File handling
    - Text file (hotel.txt)

## HOW TO RUN
   1. Install python
   2. Download the project
   3. open 'hotel_management_system.py'
   4. Run the program
